QR scanner preview Open the scanner UI for testing without starting a connection.
Nearby-first transfer WebDrop pairs nearby devices before showing file controls, keeping the main screen focused on presence.
Static web stack This prototype uses HTML, CSS, vanilla JavaScript modules, live signaling, WebRTC adapters, and streamed browser downloads with Blob fallback.
File flow Files are chosen in the send sheet, confirmed by swipe-up, sent over WebRTC DataChannel chunks, and saved by the receiving browser.
